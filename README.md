# 3D Article Keyword Word Cloud

This project is a small full-stack app that:

- Fetches the main text content of an article from a URL.
- Extracts the most important keywords using TF-IDF.
- Displays those keywords in an interactive **3D rotating word cloud** in the browser.

It is built with:

- **Backend:** FastAPI, Uvicorn, BeautifulSoup, scikit-learn (TF-IDF)
- **Frontend:** React, TypeScript, React Three Fiber, drei, three.js
